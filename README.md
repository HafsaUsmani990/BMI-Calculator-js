# BMI Calculator

A simple Body Mass Index (BMI) calculator built using HTML, CSS, and JavaScript.  
Users can enter their height and weight to calculate BMI and see their health category instantly.

## Features
- Calculates BMI based on height (cm) and weight (kg)
- Shows BMI category (Underweight, Normal, Overweight, Obesity)
- Input validation for empty or invalid values
- Clean and simple UI

## How to Use
1. Enter your height in centimeters (cm).
2. Enter your weight in kilograms (kg).
3. Click the **Calculate** button.
4. Your BMI and category will be displayed.

- CSS  
- JavaScript
